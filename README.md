# Ames Housing Price Prediction 🏠

This project aims to predict residential home prices in Ames, Iowa, using various regression techniques. It covers the full data science workflow from Exploratory Data Analysis (EDA) to model evaluation.

## 📊 Key Highlights
- **Target Accuracy:** Achieved an $R^2$ score of **0.835** on the test set.
- **Models Used:** Linear Regression, LassoCV (Regularization).
- **Diagnostics:** Performed Residual Analysis and Normality tests (Q-Q Plots) to ensure model reliability.

## 🛠️ Technologies
- Python (Pandas, NumPy, Seaborn, Matplotlib)
- Scikit-Learn (LinearRegression, LassoCV, StandardScaler)

## 📈 Results Summary
The Linear Regression model with 10 features proved to be the most effective for this dataset. Detailed analysis shows that the model generalizes well without overfitting, as evidenced by the close alignment between training and testing metrics.
